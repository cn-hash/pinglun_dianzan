抖音自动点赞脚本 - 便携式版本

使用方法：
1. 确保已安装Node.js (建议版本18+)
2. 双击 start-bot.bat 运行脚本
3. 脚本会自动打开Chrome浏览器并开始运行

功能特点：
- 智能随机点赞（基于时间段）
- 自动评论功能
- 增强键盘事件支持
- 详细日志记录

注意事项：
- 首次运行会自动安装必要的依赖
- 需要稳定的网络连接
- 建议在Windows 10/11系统上运行
